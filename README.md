# Единая Кредитная Платформа

## Описание
Цифровая платформа для совместного кредитования, объединяющая банки, МФО и инвесторов. Позволяет оформлять одну заявку на крупный кредит с автоматическим распределением суммы между несколькими кредиторами.

## 🚀 Быстрый старт

### Продакшен окружение
**Production работает на:** http://147.45.147.73:8081

```
# Запуск бэкенда
cd backend
docker compose -f docker-compose.prod.yml --env-file ./.env.dev up -d --force-recreate

# Запуск фронтенда 

cd ..
cd frontend
npm run dev
```



## 📋 Основные возможности

**Для заемщиков:**
- Одна заявка на крупный кредит
- Автоматическое распределение между кредиторами
- Единый интерфейс и трекинг статусов
- Расчет кредитоспособности и прогнозы платежей

**Для банков/МФО:**
- Диверсификация рисков
- Качественные заявки с предскорингом
- Экономия на скоринге и аналитике
- Монетизация через лидогенерацию и аналитику

## 🏗 Архитектура

**Микросервисы:**
- Скоринг и риск-менеджмент
- Оркестрация консорциума
- Интеграция банков/МФО (OpenAPI, ГОСТ-шлюзы)
- Платежный агрегатор
- Личные кабинеты (заемщик/банк)
- Аналитика и мониторинг

## 🛠 Технологии

**Backend:** Go, Python, Node.js  
**Frontend:** React, TypeScript  
**Data:** PostgreSQL, ClickHouse, Redis, S3/MinIO, Kafka  
**DevOps:** Docker, Kubernetes, CI/CD, Prometheus, Grafana, ELK  
**Security:** OAuth2, OpenID Connect, Vault, TLS+ГОСТ, 2FA, RBAC
